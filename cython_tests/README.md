tabular-predDB/cython_tests
==============

tabular predictive database cython wrapping

Instructions for building
-------------------------------------------------
    > cd /path/to/tabular-predDB/cython_tests
    > make

Running tests
---------------------------
    > # capture stdout, stderr separately
    > cd /path/to/tabular-predDB/cython_tests
    > make run >run.out 2>run.err
